# 浏览器代理切换工具 需求说明

## 模块概述
该工具是一个基于 PowerShell 的一键切换 Windows 系统代理设置的脚本。

## 核心功能清单
- **代理配置管理**: 从 `config.json` 加载代理列表。
- **系统设置修改**: 自动修改注册表 `Internet Settings`。
- **交互式列表导航**: [NEW] 支持通过 ↑/↓ 方向键选择配置，按下 `Enter` 键确认切换。
- **配置项排序**: [NEW] 支持通过 `U`/`D` 键将选中项在列表中上移或下移。
- **配置项删除**: [NEW] 支持通过 `Delete` 键删除选中的代理配置。
- **状态感知显示**: 实时检查系统当前代理设置，并在列表中以红色凸显（及 "<--- 活动" 标记）当前正在运行的配置。
- **配置持久化**: 允许保存当前系统代理为新的模板。
- **设置即时生效**: 调用 `wininet.dll` 强制浏览器刷新缓存。

## 接口/配置说明
- **ConfigFile**: 现在固定为脚本同级目录下的 `config.json`。
